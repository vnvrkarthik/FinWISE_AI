How to set up Live Integrations:

  1. Create a new Google Sheet.
  2. Go to Extensions > Apps Script.
  3. Copy the contents of [google_apps_script.js](file:///d:/vibe%20coding%20finwise/google_apps_script.js) and paste them into the script editor.
  4. Click Deploy > New Deployment. Select Web App. Set "Execute as" to "Me", and "Who has access" to "Anyone".
  5. Copy the Web App URL and paste it in the field above.
  6. Provide your Anthropic API Key. Toggle Demo Mode off, and click Save.
